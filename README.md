# Mercado y Propuesta Comercial · Cliente

Aplicación Streamlit independiente para compartir con cliente solo el bloque:

`04 · Mercado y Propuesta Comercial`

La app no incluye la navegación principal de la aplicación interna. Al abrirla, carga directamente la vista de mercado con sus pestañas internas de análisis.

## Ejecutar localmente

```bash
streamlit run app.py --server.port 8504
```

URL local probada: `http://127.0.0.1:8504`

## Despliegue

Esta carpeta puede subirse como repositorio independiente a GitHub y desplegarse en Streamlit Cloud o Render.

Archivo principal: `app.py`
